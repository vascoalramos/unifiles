Client

The Client account allows you to:

    - Review restaurants

    - See information and statistics about each restaurant

    - Create a Favorites list with your best picks.

Choose this account if you do not own any restaurants and only want to see and review restaurants of your choice.

Owner

The Owner account allows you to:

    - Add restaurants to the platform

    - Manage your existing restaurants (edit, delete)

    - Create a Favorites list with your best picks.

Choose this account if you do own restaurants and want to include them in our platform.
